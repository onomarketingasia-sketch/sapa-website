# SAPA 網站

## 頁面結構
- index.html — 主頁
- about.html — 關於我們
- programmes.html — 課程資訊
- gallery.html — 相片集
- contact.html — 聯絡我們

## CMS 後台登入
1. 去 https://你的網址/admin
2. 用受邀 Email 登入
3. 可修改：主頁內容、課程、相片集、聯絡資料

## 部署步驟
1. 上傳所有檔案到 GitHub repo
2. Netlify 連接 GitHub repo
3. Netlify → Identity → Enable
4. Identity → Services → Enable Git Gateway
5. Invite users → 輸入客人 Email
